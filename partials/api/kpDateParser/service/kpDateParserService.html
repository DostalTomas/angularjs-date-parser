  • parse(parsingDate, [customModelFormatProvider], [customViewFormatProvider]);

    Function tries to parse date using formats pipeline and if no format matches, it tries to use customViewFormatProvider if it is specified.\ Then parse date and returns it formatted using customModelFormatProvider or if it is not specified, defaultModelFormat. If no model format specified it returns ISO format.

    If no format matches, it returns null.
